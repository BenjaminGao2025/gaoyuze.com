## 欢迎来到我的博客

这是我使用 [Hugo](https://gohugo.io/) 和 [PaperMod](https://github.com/adityatelange/hugo-PaperMod) 主题搭建的个人网站的第一篇文章。

### 为什么要建这个网站？

- 📝 记录学习笔记和技术心得
- 💡 分享有价值的想法
- 🌐 建立个人品牌

### 技术栈

这个网站使用了以下技术：

| 技术 | 用途 |
|------|------|
| Hugo | 静态网站生成器 |
| PaperMod | 美观简洁的主题 |
| GitHub | 代码托管 |
| Cloudflare Pages | 全球 CDN 部署 |
